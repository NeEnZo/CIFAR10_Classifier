# CIFAR10_Classifier
2025-2026 学年 课程综合实践 I  深度学习基础——理论与实践 课程设计
## 链接
https://www.kaggle.com/competitions/2025zjutestfinal
## 说明
本竞赛作为深度学习理论与实践课程设计内容，需要大家运用在本次课程学习中所学习到的知识完成对应的要求。

具体内容为：利用本竞赛提供的cifar10数据集训练每位同学自己的神经网络，其中data_batch_1-5是训练集，test_batch是验证集。具体模型不做要求，可以自由发挥，将自己在课程学习中所学到的知识运用进来。在课程设计截止日期之后，我们会终止提交途径，以大家最后在本竞赛code部分提交的代码为准。

提交代码的测试过程如下：选择cifar10数据集中间的三类（飞机、猫、青蛙），进行测试，分别得出不同的分类正确率，以得到的分类正确率的平均值作为课程设计部分的评分基准。 课程设计部分的计分方法采用标准分计分。
